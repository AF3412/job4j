 		В СИБУР - крупную российскую нефтеперерабатывающую компанию в команду инфраструктуры управления моделями машинного обучения приглашаем на роль Java разработчика.

Задачи:

  • Разработка собственной платформы исполнения и управления моделями машинного обучения. Платформа помогает команде аналитиков данных создавать надежные сервисы из своих моделей, которые в свою очередь позволяют оптимизировать и автоматизировать различные аспекты жизни компании.
  • Создание безопасной и отказоустойчивой среды для запуска моделей.
  • Работа над улучшением продукта в сплоченной команде профессионалов.
  • Работа по методологии scrum.
  • Написание unit и интеграционных тестов.
  • Максимальная оптимизация процессов CI/CD.

Требования:

  • Знание Java 8;
  • Знание Фреймворка Spring (Core/Web/MVC);
  • Знание Jetty, Junit;
  • SQL (Postgres, MsSql), а также опыт c NoSQL базами данных
  • Сетевой стек (TCP/HTTP/SSL/REST)
  • Maven, Git
  • Опыт работы с Linux (Redhat/Centos)
  • Опыт разработки многопоточного кодом
  • Опыт разработки распределенных систем.

Будет преимуществом знание:

  • Java 9+
  • ApacheIgnite
  • BigData стек: Hadoop, Vertica, Spark
  • Ldap/ActiveDirectory
  • Protobuf/gRPC
  • ANTLR
  • GitLab, SonarQube, JFrog Artifactory

Условия:
  • Локация — Москва, офис в шаговой доступности от м. Профсоюзная.
  • Заработная плата по результатам собеседования (до 250.000 net.) + премии за эффективную работу и результат.
  • Хороший социальный пакет — ДМС и разнообразные льготы. Действуют скидки от партнеров, в том числе в части кредитования, есть возможность льготного страхования родственников.
  • Планирование карьеры — сотрудники компании имеют возможность профессионального и карьерного развития. В компании приветствуется кросс-функциональные перемещения и ротация между регионами присутствия.
  • Спорт — сотрудники имеют возможность посещать бесплатные тренировки по футболу, хоккею, йоге, пилатесу, волейболу, участвовать в корпоративных соревнованиях и семейных спортивных праздниках. Для сотрудников и членов их семей действует скидка на абонементы спортивных клубов сети World Class.
